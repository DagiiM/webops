About vLLM Deployments

All LLM models are deployed using vLLM, a high-performance inference engine optimized for large language models.

  • OpenAI-compatible API endpoints for easy integration
  • Automatic tensor parallelism for multi-GPU deployments
  • Support for quantization (AWQ, GPTQ, SqueezeLLM) to reduce memory usage
  • Continuous batching for high throughput

GPU Requirements: Most models require at least one GPU. Memory requirements vary by model size.
